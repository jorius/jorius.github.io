# 404 Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the current minimal `NotFound.tsx` with a full-page direction-b 404 experience: `BTopBar`, a massive CSS-glitched "404" hero, full i18n (EN/ES), light and dark theme support, and responsive layout. No push until the user signs off after local testing.

**Architecture:** `NotFound` becomes a top-level standalone route (like `Writing`) rather than a child of the `/*` catch-all, giving it full control over its chrome. The legacy pages (`/about`, `/portfolio`, `/contact`) are extracted into a `LegacyLayout` component that renders the old `Header`/`Footer` via React Router's `<Outlet>`. The glitch animation is pure CSS keyframes injected via a `<style>` JSX element so all three ghost layers stay frame-perfectly in sync.

**Tech Stack:** React 19, React Router DOM 7, react-i18next, inline styles + `useBTheme()` tokens, `<style>` JSX for keyframes, `useIsMobile()` for responsive breakpoints.

### Task 4: Restructure routes in `App.tsx`

**Files:**
- Modify: `src/App.tsx`

Replace the `/*` catch-all (with inline element + inner `<Routes>`) with a `LegacyLayout` layout route and a top-level `*` route for `NotFound`.

- [ ] **Step 1: Replace the full contents of `src/App.tsx`**

```tsx
// packages
import { BrowserRouter as Router, Routes, Route, Navigate } from 'react-router-dom';

// components
import LegacyLayout from './components/LegacyLayout';
import { ScrollToTop } from './components/ScrollToTop';

// pages
import About from './pages/About';
import Contact from './pages/Contact';
import Home from './pages/Home';
import NotFound from './pages/NotFound';
import Palette from './pages/Palette';
import Pgp from './pages/Pgp';
import Portfolio from './pages/Portfolio';
import Writing from './pages/Writing';

function App() {
  return (
    <Router>
      <ScrollToTop />
      <Routes>
        {/* Standalone routes — render their own chrome (header/footer baked in). */}
        <Route path="/" element={<Home />} />
        <Route path="/writing" element={<Writing />} />
        <Route path="/writing/:slug" element={<Writing />} />
        <Route path="/read/:slug" element={<Navigate to="/writing" replace />} />
        <Route path="/pgp" element={<Pgp />} />
        <Route path="/palette" element={<Palette />} />

        {/* Legacy routes — wrapped in old Header/Footer until each page is redesigned. */}
        <Route element={<LegacyLayout />}>
          <Route path="/about" element={<About />} />
          <Route path="/portfolio" element={<Portfolio />} />
          <Route path="/contact" element={<Contact />} />
        </Route>

        {/* 404 — standalone, uses BTopBar. Must be last. */}
        <Route path="*" element={<NotFound />} />
      </Routes>
    </Router>
  );
}

export default App;
```
